```
配置github的ssh密钥:
输入 cd ~/.ssh
ssh-keygen -t rsa -C "youremail@youremail.com"
然后会在 .ssh 目录生产两个文件：id_rsa和id_rsa.pub
复制ssh key到github：
On the GitHub site Click “Settings” > Click “SSH and GPG Keys” > Click “New SSH key”
```
